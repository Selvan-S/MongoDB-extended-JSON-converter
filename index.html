Supported MongoDB Extended JSON Types

The converter handles all MongoDB-specific types including complex nested structures

Numeric Types

  • $numberInt → Number
  • $numberLong → Number
  • $numberDouble → Number
  • $numberDecimal → Number

Date & Time

  • $date → ISO 8601 String
  • Supports both string and numeric formats
  • Handles negative timestamps (pre-1970)

Identifiers

  • $oid → String
  • $binary → Base64 String
  • $uuid → String

Special Types

  • $minKey → "MinKey"
  • $maxKey → "MaxKey"
  • $undefined → null
  • $regex → RegExp object
